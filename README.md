
# 🎵 Music Player

A responsive and interactive Music Player built using HTML, CSS, and JavaScript as part of the Frontend Development Internship at CodeAlpha.

## 🚀 Features

* ▶ Play and Pause Songs
* ⏮ Previous Song Navigation
* ⏭ Next Song Navigation
* 🎵 Dynamic Playlist
* 🔎 Search Songs by Title or Artist
* ❤️ Favorites System
* 💾 Favorites Saved Using Local Storage
* 📊 Interactive Progress Bar
* ⏱ Song Duration and Current Time Display
* 🔊 Volume Control
* 🌙 Dark Mode Support
* 🔄 Auto Play Next Song
* 📱 Responsive User Interface

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Use

1. Open the application in your browser.
2. Select a song from the playlist.
3. Use Play/Pause controls to manage playback.
4. Navigate songs using Next and Previous buttons.
5. Search songs using the search bar.
6. Add songs to Favorites by clicking the heart icon.
7. Adjust volume using the volume slider.
8. Switch between Light and Dark Mode.
